# Data

The raw Telco Customer Churn dataset is not included in this repository.

To re-run the full project locally, download the Telco Customer Churn CSV and place it in this folder using the expected filename:

```text
data/WA_Fn-UseC_-Telco-Customer-Churn.csv
```

The notebooks also support the original Kaggle input paths when run on Kaggle.

Generated project outputs are stored in the `outputs/` folder.
