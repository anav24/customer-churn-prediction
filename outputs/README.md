# Outputs

This folder contains generated CSV outputs from the customer churn notebooks. These files are included so the project findings can be reviewed without re-running every notebook, and they can be regenerated by running the notebooks in order.

Included outputs:

- `telco_churn_cleaned.csv`: cleaned customer-level dataset created in the EDA notebook
- `contract_tenure_segments.csv` and `customer_tenure_segments.csv`: contract and tenure churn segment summaries
- `internet_payment_segments.csv`: internet service and payment method segment summary
- `support_segments.csv`: online security and tech support segment summary
- `risk_factor_summary.csv`: churn rate by number of risk factors
- `key_churn_segments.csv`: final high-risk customer segment summary
- `model_comparison_results.csv`: baseline and model comparison metrics
- `logistic_regression_threshold_results.csv`: threshold tuning results
- `final_model_summary.csv`: selected model metrics
- `churn_recommendations.csv`: recommendation table
- `recommendation_priority.csv`: prioritized business actions

The original raw dataset is not stored in this repository. See `data/README.md` for dataset source information.
